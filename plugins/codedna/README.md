# CodeDna Claude Plugin

This plugin packages CodeDna for Claude Code.

It provides:

- a reusable `codedna` skill in `skills/codedna/`

Prerequisite:

- `codedna` must be installed and available in `PATH`

Install the CLI first:

```bash
cargo install codedna
```

If you are using the marketplace from this repository:

```text
/plugin marketplace add crafteraadarsh/codedna
/plugin install codedna@codedna-marketplace
```
