# CodeDna Skill Overview

This root file is a quick pointer, not the actual Codex skill package.

Use:

- `skill/codedna/` for the valid Codex skill package
- `skills/codedna-analyzer/SKILL.md` for the full cross-agent reference and examples
- `AGENTS.md`, `CLAUDE.md`, `GEMINI.md`, and `.cursorrules` for repo-local agent bootstrap files

Quick start:

```bash
cargo install --git https://github.com/crafteraadarsh/codedna
codedna analyze .
codedna analyze https://github.com/vercel/next.js
codedna json . --compact
```
